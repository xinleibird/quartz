> [!summary]
>「松散模式」也就是 JavaScript 历史上遗留的，在 JavaScript 早期版本采用的「原始模式」。也可以看做是与 [[JavaScript/Mode/strict|严格模式]] 相反的「非严格模式」。

> [!quote] 相对概念
> 需要注意的是，「松散模式」并不是一个官方术语，而仅仅是一个与 [[JavaScript/Mode/strict|严格模式]] 相对的概念。
